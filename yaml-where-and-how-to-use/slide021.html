Perl YAML Processors

  • All the Perl Modules behave slightly different
  • YAML.pm:
    Based on YAML 1.0 and has lots of problems with valid 1.1/1.2 syntax
  • YAML::Syck:
    Based on libsyck/YAML 1.0 and also has problems, including segfaults
  • YAML::XS:
    Based on libyaml/YAML 1.1 and will handle most YAML 1.1/1.2
  • YAML::Tiny:
    Implements a subset of YAML (1.0/1.1?)
  • YAML::PP:
    Aims for YAML 1.2, but is still in development
