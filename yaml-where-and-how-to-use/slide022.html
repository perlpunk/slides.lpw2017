Some notes about booleans

  • Most JSON processors will load Booleans into JSON::PP::Boolean objects
{ "true": true, "false": false }
# JSON::XS # JSON::PP # JSON.pm # Cpanel::JSON::XS # Mojo::JSON $VAR1 = { 'true' => bless( do{\(my $o = 1)}, 'JSON::PP::Boolean' ), 'false' => bless( do{\(my $o = 0)}, 'JSON::PP::Boolean' ) };
